Project Information

このサイトは大学の課題として制作された、次世代型Vlog共有プラットフォームのデモ版です。

🛠 使用技術

  • HTML5 / CSS3 (Vanilla)
  • JavaScript (ES6+)
  • Firebase: Firestoreによるリアルタイム共有
  • Leaflet.js: 旅の軌跡を可視化するマップ機能
  • LocalStorage: ログイン不要の個人投稿管理

✨ 注目機能

  • ジャーニー・マップ: 投稿詳細から旅のルートを地図上で確認できます。
  • SNSフィード: 他のユーザー（デモデータ含む）の投稿を一覧できます。
  • クイック投稿: アカウント作成不要で、即座に思い出を共有可能です。
