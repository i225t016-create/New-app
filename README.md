# VlogFolio

美しくモダンなVlog公開用ポートフォリオ。HTML, CSS, JavaScriptのみで構築されており、GitHub Pagesで簡単に公開できます。

## 特徴
- **モダンなデザイン**: ダークテーマ、グラスモフィズム、滑らかなアニメーション。
- **データ駆動**: `data/vlogs.json`を編集するだけでコンテンツの更新が可能。
- **レスポンシブ**: PC、タブレット、スマホのすべてのデバイスに対応。
- **ビデオプレイヤー**: モーダルウィンドウによるスムーズな動画再生。

## 公開方法 (GitHub Pages)
1. このリポジトリをGitHubにプッシュします。
2. GitHubリポジトリの「Settings」タブを開きます。
3. 左サイドバーの「Pages」をクリックします。
4. 「Build and deployment」セクションの「Branch」で `main` (または `master`) ブランチを選択し、「Save」をクリックします。
5. 数分後、提供されたURLでサイトが公開されます。

## 開発
- `index.html`: サイトの構造
- `css/style.css`: デザインとスタイル
- `js/app.js`: 動的機能とロジック
- `data/vlogs.json`: 動画データ
