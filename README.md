# VlogFolio - 次世代型Vlog共有プラットフォーム（大学課題デモ版）

## 1. プロジェクト概要
本プロジェクトは、旅行や日常の思い出を「点（写真）」ではなく「線（軌跡）」として記録・共有することを目的としたWebアプリケーションです。
既存のSNSの利便性と、ジャーニー・マップ（旅の軌跡）による視覚的なストーリーテリングを融合させています。

## 2. 注目機能（差別化ポイント）
- **ジャーニー・マップ**: 地図ライブラリ「Leaflet.js」を使用。タイムラインに記録された各地点を座標データとして扱い、地図上にルートを描画します。
- **SNSフィード機能**: Firebase Firestoreを活用したリアルタイム共有。他ユーザーの投稿を即座に閲覧可能です。
- **アカウントレス・投稿システム**: 会員登録の障壁をなくし、ニックネームだけで即座に投稿。LocalStorageを活用し、個人デバイス上での「自分の投稿」管理を実現。
- **BeReal/Setlog風UI**: 没入感を高める黒基調のミニマルデザインと、直感的なナビゲーション。

## 3. 使用技術（Tech Stack）
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla ES6+)
- **Backend**: Firebase (Cloud Firestore)
- **Map Library**: Leaflet.js
- **Deployment**: GitHub Pages

## 4. デモの体験方法
1. **ホーム（みんなの投稿）**: 
   - 既存のサンプルデータや、他のユーザーの投稿を閲覧できます。
   - 投稿をタップすると詳細が開きます。上部の地図でルートを確認してください。
2. **新規投稿**: 
   - 下部の「➕」ボタンから、新しい思い出を投稿できます。
   - 画像URL、タイトル、場所、詳細を入力して公開してください。
3. **自分の投稿**: 
   - 下部の「👤」ボタンから、自分がこのブラウザで投稿した内容を確認できます。

## 5. 開発者より
本アプリは「思い出をより鮮明に、かつ手軽に」残すためのUI/UXを追求しました。特に地図とタイムラインをシームレスに結合させることで、後から振り返った際の没入感を高めています。
